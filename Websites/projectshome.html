Project Name: Mia's Shop.

This project consists of a fictional coffee shop, found in Ireland. The goal is to show my understanding of Cascading Style Sheets, HTML and JavaScript. While the JavaScript aspect is minimal at the moment, the final goal is to have it work like a web store, allowing the user to add, delete and check out items purchased.

Project Name: RCG (Random Color Generator)

This project consists of understanding of lists, random numbers and adding each one to a new variable, using JavaScript. after getting a random hex value, it is then pushed as the background and the random color is displayed as the background. This is a very simple HTML and CSS because the main focus is using JavaScript.
